PiBrush -- a Paint Brush for the Pi

This tree contains the code for a simple daemon for PiBrush, a virtual
paint-brush for the Raspberry Pi.

The daemon implementation is based on SimBaMon -- the simple battery monitor
daemon -- see ../simbamon/README.txt

For more information surf on over to http://pi.gate.ac.uk/pages/pibrush.html
or see the quick-start.txt document.
