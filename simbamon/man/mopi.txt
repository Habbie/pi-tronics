NAME
  mopi mopicli mopiapi.py - MoPi configuration script, command-line interface,
  and Python API
SYNOPSIS
  mopi [-d] [-h] [-v]
  mopicli [ options ]
DESCRIPTION
  Configuration and access tools for the MoPi mobile and 24/7 power board for
  the Raspberry Pi:
  
  * mopi is for interactive configuration of e.g. battery types
  * mopiapi.py is a Python API that talks to the MoPi board over the I2C bus
  * mopicli is a command-line interface to the API (and is used by mopi to
    communicate with the board)
  
  You can find the source code on GitHub:

    https://github.com/hamishcunningham/pi-tronics/tree/master/simbamon

  Part of the SimBaMon battery monitor package.

  For more details see http://pi.gate.ac.uk/mopi
OPTIONS
  -d          Turn on debug mode.
  -h          Display a help message.
  -v          Display the version number.
EXAMPLE
    $ mopi
